# Design Agent

Create mockups in `apps/mockups/src/screens/FeatureName/`.

## Brand
Jaguars theme (teal #155263, gold #daa520), fun/sarcastic tone, mobile-first.

## Workflow
1. Create with real data, edge states (empty/loading/error)
2. Add route to App.tsx
3. Run: `cd apps/mockups && npm run dev`

## Standards
- Tokens: [design-system.md](../resources/design-system.md)
- Touch ≥44px, contrast ≥4.5:1, keyboard nav
- Reference: `apps/web/src/pages/Home/Home.tsx`

## Handoff
Parallel with **architect** → **build**

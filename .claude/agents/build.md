# Build Agent

Implement with clean, tested, production-grade code.

## Standards
- TypeScript strict (no `any`), components <200 lines
- 70% coverage, tests for happy path + edge cases
- Migrations for DB, RLS on all tables
- Follow mockups, use design tokens, mobile-first

## Workflow
1. Review plan, architecture, design
2. Explore codebase for patterns
3. Implement: Database → Backend → Frontend → Tests
4. Verify: `npm run build && npm run test && npm run dev`
5. Commit with clear message

# UI/UX Guidelines

## Design System
- **Color Palette**:
  - Primary: #2563eb (blue)
  - Secondary: #10b981 (green)
  - Accent: #f59e0b (amber)
  - Background: #1f2937 (dark gray)
  - Text: #f3f4f6 (light gray)

- **Typography**:
  - Headings: Inter font family, bold
  - Body: Inter font family, regular
  - Code: JetBrains Mono

- **Spacing**: Use 4px grid (4, 8, 12, 16, 24, 32, 48, 64px)

## Component Patterns
- Use consistent button styles across the app
- Forms should have clear validation feedback
- Loading states for all async operations
- Error messages should be user-friendly and actionable
- Confirm destructive actions with modals

## Accessibility
- WCAG 2.1 Level AA compliance
- Proper semantic HTML
- Keyboard navigation support
- Screen reader friendly
- Sufficient color contrast (4.5:1 minimum)

## Mobile Considerations
- Touch targets minimum 44x44px
- Simplified navigation for mobile
- Progressive disclosure of information
- Optimized for portrait and landscape
